Windows Server 2003不能加载移动硬盘

发表于 2013-10-01 | 分类于 未分类 ， 网络技术

这个问题表现的比较突出，想必会有很多朋友在使用Server系统时连接好移动硬盘后却在“我的电脑”中找不到相应的硬盘盘符，但是到磁盘管理里面却可以看到。这是因为Server系统没有自动加载磁盘分区，这时我们可以在磁盘管理里面手动给移动硬盘分配上盘符，然后就可以在“我的电脑”里看到移动硬盘了。可是当我们下次换了一块硬盘接上去后，发现还是没有分配盘符。这时我们就可以使用diskpart命令来使用系统自动加载移动硬盘分区了。

打开“命令提示符”窗口，输入diskpart后显示DISKPART>状态，我们再输入automount enable然后回车，重新启动计算机就算下次换了其他的移动硬盘也可以系统自动为它分配盘符了。

# Windows 2003不能加载移动硬盘
电脑每次开机都要按F1才能启动解决办法
