hexo博客next主题html页面压缩

发表于 2017-12-12

由于next主题没有压缩md文件，导致最终生成的Md文件没有删除空行。所以可能用gulp来布压缩一下，由于本人设置生成的文件最终是以html结尾，这样方便搜索引擎收录，压缩也方便。
单独建了一个html文件夹放生成的html文件，也就是每篇博客最终是一个html文件，这个后缀只需要在_config.yml 文件里面的permalink 关键字后面设置成permalink: html/:title.html 就可以了。

1 用npm安装gulp和gulp-minify-html 两个插件

2 在根目录下建立一下gulpfile.js文件

3 写入以下代码：

var gulp = require('gulp'),
    uglify = require("gulp-uglify"),
    miniHtml = require("gulp-minify-html");

gulp.task('js', function () {
    gulp.src(['public/js/src/schemes/*.js','public/js/src/*.js']) // 要压缩的js文件
    .pipe(uglify())  //使用uglify进行压缩,更多配置请参考：
    .pipe(gulp.dest('public/js/src/schemes/ys')); //压缩后的路径
});


gulp.task('html', function () {
    gulp.src(['public/html/*.html']) // 要压缩的js文件
    .pipe(miniHtml())  //使用uglify进行压缩,更多配置请参考：
    .pipe(gulp.dest("public/html/")); //压缩后的路径
});
