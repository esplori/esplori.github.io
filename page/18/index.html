中国中央政府 2014 年采购为什么禁止设备预装 Windows 8 系统？

发表于 2014-07-30 | 分类于 未分类 ， 网络精选

在我国政府的采购政策是，采购时选择可信厂商，原则上除非必需，不选择国外厂商。但是这个事情在2001年被打破，因为2001年美国一票厂商都出台了源代码共享协议。

:: 微软公司推出“政府安全计划” 开放源代码 ::（每次看见我那NC果粉软黑学长发言就觉得好笑，说微软对政府开放源代码是我编的……我来把新华网的链接补上）

以微软为例，如果某国政府购买了微软的产品，微软会随产品同时交付并允许客户保留97%的源代码，剩余3%的代码涉及商业技术机密，如NT内核代码、Office格式支持和排版引擎、CRM和ERP的BI部分。对于这3%的源代码，微软允许政府机构派遣人员到微软总部的Build实验室去现场审查所有代码，并且，允许用户全程监督的情况下指定经过审查的源代码副本进行Build和封装。换句话说，整个系统的安全性完全受客户自己掌控。具有类似协议的美国公司还包括Adobe、IBM、SAP、Symantec、Mcafee等等。换句话说，任何形式的以国家安全名义的排斥持有这些协议的公司的产品，在不怀疑政府外派专家的前提的条件下，都是实实在在的幌子。
