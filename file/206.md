前段时间重装windows后，github就用不了，现在想重装，一直遇到一个问题。

提示an error occured attempting to install GitHub

于是乎在chrome上打开http://github-windows.s3.amazonaws.com/GitHub.application

成功下载github.application,安装，仍然出现如下问题

于是把链接 http://github-windows.s3.amazonaws.com/GitHub.application 中的http改为https，然后在IE浏览器中打开即可安装。

祝好运！
