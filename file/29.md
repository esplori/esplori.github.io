cmd命令下：ntsd -c q -pn winlogon.exe

如果是windows2000，则需要得到winlogon.exe的pid，可以用任务管理器得到其pid。
然后用ntsd -c q -p pid 即可让电脑蓝屏～
windows xp也可以用这个命令

&nbsp;

原因：ntsd -c q -pn winlogon.exe 的意思是结束winlogon.exe 这个进程。winlogon.exe用户登陆程序，管理用户登录和退出 所以你结束了 你系统就会蓝屏，死机。
