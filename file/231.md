网上都说是把
<pre>&lt;?php if(function_exists('wp_pagenavi')) { wp_pagenavi(); } ?&gt;</pre>
替换掉
<pre>&lt;div class="navigation"&gt;
&lt;p&gt;&lt; span&gt;&lt;?php next_posts_link(’? 较旧文章’)?&gt;&lt;/span&gt;
&lt;span class="nextlink"&gt;&lt;?php previous_posts_link(’较新文章 ?’)?&gt;&lt;/span&gt;&lt;/p&gt;
&lt;/div&gt;</pre>
这段代码，可在最新的主题偏偏没有这段语句，而是
<pre>&lt;?php twentyeleven_content_nav( 'nav-below' ); ?&gt;</pre>
替换之后终于能到那段显示页码的导航条了。

效果：

&nbsp;

以下是实现的两个方法：

方法一：

把archive.php、index.php、search.php这三个文件中的
<pre>&lt;?php twentyeleven_content_nav( 'nav-below' ); ?&gt;</pre>
全部用以下代码替换掉：
<pre>&lt;?php if(function_exists('wp_pagenavi')) { wp_pagenavi(); } ?&gt;</pre>
方法二：

找到functions.php中的
<pre>&lt;h3&gt;&lt;?php _e( 'Post navigation', 'twentyeleven' ); ?&gt;&lt;/h3&gt;
&lt;div&gt;&lt;?php next_posts_link( __( '&lt;span&gt;&amp;larr;&lt;/span&gt; Older posts', 'twentyeleven' ) ); ?&gt;&lt;/div&gt;
&lt;div&gt;&lt;?php previous_posts_link( __( 'Newer posts &lt;span&gt;&amp;rarr; &lt;/span&gt;', 'twentyeleven' ) ); ?&gt;&lt;/div&gt;</pre>
用以下代码替换：
<pre>&lt;?php if(function_exists('wp_pagenavi')) { wp_pagenavi(); } ?&gt;</pre>
