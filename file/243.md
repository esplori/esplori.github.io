WINDOWS 2003的安全设置选项默认是不允许修改安全等级的。在浏览器的“工具&gt;Internet选项&gt;安全”这个选项卡下，安全等级总是“高”。而且不允许修改。

&nbsp;

修改分两步：

1.在"控制面板&gt;添加或删除程序&gt;添加/删除WINDOWS组件"里面，将"Internet Explorer 增强的安全选项"前面的对勾去掉。

然后下一步。

2.在"开始&gt;运行"里面输入“regedit”，打开注册表。找到“HEKY_CURRENT_USER&gt;Microsoft&gt;Windows&gt;CurrentVersion&gt;Internet Setting&gt;Zones&gt;3”，将右边列表的“MinLevel”键值设置为10000（十六进制），默认为12000，“RecommendedLevel”键值设置为10000（十六进制）。
