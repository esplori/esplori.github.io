wordpress 默认主题的首页的文章显示的都是全文，用户体验不是很好，下面是把全文改为摘要的方法，只显示一部分内容 ，对于用户来说可以更快的找到重点。

打开content.php,找到如下代码：
`&lt;?php
the_content( __( 'Continue reading &lt;span&gt;→&lt;/span&gt;', '当前主题' ) );
?&gt;`

将它修改为：

`&lt;?php
if(!is_single()) {
the_excerpt();
} else {
the_content(__('(more…)'));
}
?&gt;`
