感谢您为您的 Apple ID (1•••••@qq.com) 启用双重认证。
双重认证是一道额外安全防线，用来防止未经授权的用户访问您的帐户；并保障您储存在 Apple 上的照片、文稿以及其他数据的安全。
从现在开始，每当您在新设备或浏览器上登录 Apple ID，您将需要输入密码和一个发送至您其他设备或者受信任电话号码的六位数验证码来验证您的身份。如需了解更多，请阅读常见问题解答。
更多安全信息
如果您未启用双重认证并认为有未经授权的用户访问了您的帐户，请使用下方链接返回之前的安全设置。
关闭双重认证
此链接和您的 Apple ID 安全提示问题将于2016年11月9日后失效。此后，您只能使用您的密码和六位数验证码登录 Apple ID 帐户页面才能关闭双重认证。
